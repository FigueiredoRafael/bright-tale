01 · Pesquisa de mercadoMarket research

O que o mercado está fazendo em 2026.

What the market is doing in 2026.

Três frentes de pesquisa foram conduzidas antes da auditoria da pipeline: geração de thumbnails, estrutura de roteiros e automação de Shorts/B-roll. Os achados estão consolidados abaixo com fontes.

Three research tracks were conducted before the pipeline audit: thumbnail generation, script structure, and Shorts/B-roll automation. Findings are consolidated below with sources.

Geração de Thumbnails

Thumbnail Generation

3 fontes 3 sources

Estado da arte em 2026: ferramentas como Thumbmagic, ThumbnailCreator e YTZolo geram thumbnails via IA com predição de CTR antes da publicação. O YouTube já testa servir thumbnails diferentes para segmentos diferentes de audiência (A/B automático nativo).

State of the art in 2026: tools like Thumbmagic, ThumbnailCreator, and YTZolo generate thumbnails via AI with CTR prediction before publishing. YouTube is already testing serving different thumbnails to different audience segments (native automatic A/B).

Padrão de "dualidades": geração de 3–5 variações de conceito com emoção definida (curiosity / shock / intrigue), paleta de cor, composição e texto overlay. A/B testing é feito pelo criador com base nos conceitos.

"Dualities" standard: generation of 3–5 concept variations with defined emotion (curiosity / shock / intrigue), color palette, composition, and text overlay. A/B testing done by the creator based on concepts.

Relevância para a pipeline: o campo thumbnail_ideas[] já existe no output do BC_VIDEO com exatamente esse formato — 3–5 conceitos, emoção, paleta e composição. Está implementado e alinhado com o padrão de mercado.

Pipeline relevance: the thumbnail_ideas[] field already exists in BC_VIDEO output with exactly this format — 3–5 concepts, emotion, palette, and composition. It's implemented and aligned with market standard.

Estrutura de Roteiros

Script Structure

5 fontes 5 sources

Diferença crítica 2026: roteiros profissionais distinguem dois documentos separados — teleprompter script (leitura limpa para apresentador ou TTS, sem cues de produção) e editor script (guia técnico com A-roll, B-roll, SFX, texto overlay e transições). O agente BC_VIDEO já gera os dois.

Critical 2026 distinction: professional scripts separate two documents — teleprompter script (clean reading for presenter or TTS, no production cues) and editor script (technical guide with A-roll, B-roll, SFX, text overlays, and transitions). The BC_VIDEO agent already generates both.

Padrão de velocidade: 130–160 palavras por minuto para narração natural. Vídeo de 8 min = ~1.200–1.280 palavras no teleprompter. A regra de escalonamento por target_duration_minutes já existe no agente, mas o campo não está declarado formalmente no inputSchema (Gap G2).

Pace standard: 130–160 words per minute for natural narration. 8-min video = ~1,200–1,280 words in teleprompter. The scaling rule by target_duration_minutes already exists in the agent, but the field isn't formally declared in the inputSchema (Gap G2).

Canais dark/faceless: script deve ser escrito para performance vocal, sem referências à presença física. Linguagem de IA como "Como você pode ver aqui" é incompatível com TTS. Exige channel_type=dark no input do agente para adaptar o tom (Gap G3).

Dark/faceless channels: script must be written for vocal performance, without references to physical presence. AI language like "As you can see here" is incompatible with TTS. Requires channel_type=dark in agent input to adapt tone (Gap G3).

Automação de Shorts e B-roll

Shorts and B-roll Automation

4 fontes 4 sources

Pipelines open-source em 2026: projetos como ShortGPT e AI-Youtube-Shorts-Generator fazem highlight detection via LLM, transcrição Whisper, recorte vertical 9:16 automático e score de viralidade antes de exportar — tudo a partir de um vídeo longo já existente.

Open-source pipelines in 2026: projects like ShortGPT and AI-Youtube-Shorts-Generator do LLM highlight detection, Whisper transcription, automatic 9:16 vertical cropping, and virality scoring before export — all from an existing long-form video.

Implicação para nossa pipeline: hoje os Shorts são derivados do BC_CANONICAL_CORE, independente do vídeo. O padrão de mercado é derivar de capítulos já gerados. O mapper mapVideoOutputToShortsInput() que precisamos criar (Gap G7) está alinhado com esse padrão.

Implication for our pipeline: today Shorts are derived from BC_CANONICAL_CORE, independent of the video. The market standard is to derive from already-generated chapters. The mapVideoOutputToShortsInput() mapper we need to create (Gap G7) aligns with this standard.

B-roll com Ken Burns + voice ducking são padrão em canais dark. Animação de B-roll (zoom/pan automático) + trilha sonora com duck automático na narração estão nos pipelines abertos de 2026. Isso é escopo de pós-edição — fora da nossa pipeline de roteiro.

Ken Burns B-roll + voice ducking are standard in dark channels. B-roll animation (auto zoom/pan) + soundtrack with automatic duck on narration are in 2026 open pipelines. This is post-editing scope — outside our script pipeline.

G1 — VideoStyleSelector ausente do DraftEngine

G1 — VideoStyleSelector missing from DraftEngine

Crítico Critical

O VideoStyleSelector existe apenas no ProductionForm (pipeline YAML legado). O DraftEngine — fluxo ativo — nunca lê nem envia video_style_config. O agente recebe o objeto vazio em todos os vídeos gerados hoje.

The VideoStyleSelector only exists in ProductionForm (legacy YAML pipeline). The DraftEngine — the active flow — never reads or sends video_style_config. The agent receives an empty object for every video generated today.

G2 — production_params não declarado no inputSchema

G2 — production_params not declared in inputSchema

Alto High

O DraftEngine envia target_duration_minutes corretamente, e a regra existe no agente: "If production_params.target_duration_minutes is provided, scale teleprompter_script." Mas o campo não está declarado no inputSchema do seed — o LLM não o vê formalmente e o ignora na maioria dos casos.

The DraftEngine correctly sends target_duration_minutes, and the rule exists in the agent: "If production_params.target_duration_minutes is provided, scale teleprompter_script." But the field is not declared in the seed's inputSchema — the LLM doesn't see it formally and ignores it in most cases.

G3 — channel_type dark/presenter not signaled

O agente não distingue canal dark de canal com apresentador — o teleprompter_script gerado tem cues visuais ([look at camera]) mesmo em canais 100% faceless.

The agent can't distinguish dark from presenter channels — generated teleprompter_script includes visual cues ([look at camera]) even for 100% faceless channels.

G4 — Lower thirds missing from editor_script

G5 — camera_count ausente no video_style_config

G5 — camera_count missing from video_style_config

Médio Medium

G6 — TTS não conectado ao teleprompter_script

G6 — TTS not connected to teleprompter_script
