Architecture

Use next-themes for system-preference detection + toggle. All colors defined as CSS custom properties in globals.css with :root (light) and .dark (dark) scopes. Components reference tokens, never raw hex values.

Key Design Decisions

  • Glassmorphism cards: Semi-transparent backgrounds with backdrop-filter: blur() + subtle white border in dark mode. Clean white cards with soft shadows in light mode.
  • Glow accents: Top-edge gradient lines on featured cards (brand teal). Pulsing glow on health indicators.
  • Token system: ~20 semantic CSS vars (--admin-bg, --admin-card, --admin-border, --admin-text-primary, etc.) scoped under :root / .dark.
  • Brand colors stay constant: Teal, orange, purple don't change between themes — only surfaces and text adapt.
  • next-themes: ThemeProvider wrapping admin layout, useTheme() for toggle button in header.
