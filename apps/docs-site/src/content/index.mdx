## O que o BrightTale gera

| Mídia | Variantes | Status |
|---|---|---|
| **Blog** | SEO + afiliados + WordPress | ✅ Implementado |
| **Vídeo YouTube** | Canal normal, dark, cursos | ✅ Parcial |
| **Shorts** | YouTube Shorts / Reels | ✅ Implementado |
| **Podcast** | Roteiro + talking points | ✅ Implementado |
| **Engagement** | CTAs, hooks sociais | ✅ Implementado |

## Pipeline

```
Brainstorm → Research → Production → Review → Publish
   Agent 1     Agent 2     Agent 3     Agent 4    (WordPress/YouTube)
```

## Tech Stack

| Camada | Tecnologia |
|---|---|
| Framework | Next.js 16 (App Router) + React 19 |
| Linguagem | TypeScript |
| Database | Supabase (PostgreSQL) |
| Validação | Zod |
| UI | shadcn/ui + Tailwind CSS 4 |
| Imagens IA | Gemini Imagen |
| Testes | Vitest 4 |
