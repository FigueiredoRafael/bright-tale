# Podcasts

Roteiros de podcast com talking points e perguntas para convidados.

## Funcionalidades

- Título e descrição do episódio
- Intro hook (abertura que segura o ouvinte)
- Talking points com timings
- Ângulo pessoal do host
- Perguntas para convidado
- Outro (encerramento)
- Export para markdown
