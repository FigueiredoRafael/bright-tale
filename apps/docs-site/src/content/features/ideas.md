# Ideias

Biblioteca de ideias com detecção de similaridade e tags.

## Funcionalidades

- Criar ideias manualmente ou a partir do brainstorm
- Detecção automática de similaridade ao criar
- Filtros: veredito, tipo de origem, tags, público/privado
- Busca por texto
- Contador de uso por ideia
- Suporte a formato legado (30 dias de backward compatibility)

## Campos Principais

| Campo | Descrição |
|---|---|
| `title` | Título da ideia |
| `core_tension` | Tensão central |
| `target_audience` | Público-alvo |
| `verdict` | `viable` ou `weak` |
| `source_type` | brainstorm, manual, import |
| `tags` | Array de tags |
| `is_public` | Visível para outros usuários |
