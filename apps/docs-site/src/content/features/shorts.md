# Shorts

YouTube Shorts / Reels — vídeos curtos de 15-60 segundos.

## Funcionalidades

- 3-5 clips por projeto
- Cada clip: hook, corpo, CTA, duração
- Captions e transições
- Export para markdown

## Estrutura de um Clip

| Campo | Descrição |
|---|---|
| `hook` | Abertura que captura atenção (3s) |
| `body` | Conteúdo principal |
| `cta` | Call-to-action de encerramento |
| `duration` | 15-60 segundos |
| `captions` | Legendas habilitadas |
| `transition` | Tipo de transição |
