# WordPress

Integração para publicar blogs diretamente no WordPress.

## Funcionalidades

- Configurar múltiplos sites WordPress
- Publicar como draft ou ao vivo
- Selecionar categorias e tags do WordPress
- Blog content puxado automaticamente do Production stage
- Armazena `wordpress_post_id` no blog draft

## Fluxo

1. Configurar site WordPress em Settings (URL + credenciais)
2. No blog draft, clicar "Publicar no WordPress"
3. Selecionar categorias e tags
4. Escolher status (draft ou publish)
5. Receber URL do post publicado
